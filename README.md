# Superstore Retail Sales Analytics Dashboard

## Project Overview
This project serves as a comprehensive business intelligence solution to analyze retail performance. By leveraging the Superstore dataset, I developed an interactive dashboard that transforms raw transaction data into visual insights regarding profitability, regional sales trends, and product category performance.

## Tech Stack
- Data Modeling & BI: Power BI Desktop
- Data Analysis Expressions (DAX): Custom measures for Profit Margin and Average Order Value.
- Data Visualization: Clustered bar charts, donut charts, and scatter plots for multi-dimensional analysis.

## Key Dashboard Insights
- Performance KPIs: Tracked critical business metrics (Total Sales, Profit Margin %, AOV) to monitor financial health.
- Category Profitability: Identified high-margin vs. low-margin product categories to assist in portfolio optimization.
- Logistics Efficiency: Analyzed shipping preferences (`Ship Mode`) to understand operational patterns.
- Discount Analysis: Used scatter plots to visualize the correlation between discount levels and profitability, helping define an optimal pricing strategy.
- Interactive Filtering: Implemented category-level slicers to allow users to drill down into specific departmental performance.
